# tienda-fotos
Pagina para comprar fotos variadas

Configuraciones del aplicativo
Versión de Node: v14.21.1
Versión de Sails: 1.5.3
Versión de NPM: 8.19.3
Versión de postgres: 6.15

Instalar el aplicativo
1- Realizar la restauración de la base de datos con apoyo del archivo de respaldo dump_tienda_virtual
2- Configurar el acceso al servidor de base de datos, este se configura en el archivo config>datastores.js
3- Realizar la instalación de dependencias con npm install
4- Ejecutar el comando nodemon app.js para iniciar el programa
